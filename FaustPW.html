Faust
Theoretical Project

To the left you will find the overhead light plot for the Faust project for Advanced Lighting III with collaboration from scenic and costume students. The theater for this project is the Patricia Corbett Theatre and there were no lighting inventory restrictions given for this project.

Below are other elements that were required for this assignment, such as a Color Angle Concept, Magic Sheet, Area Layout, Hanging Section, Paperwork Package, and 5 3D renderings done in Augment3d depicting various parts of the opera.
