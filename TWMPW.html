The Whipping Man
Theoretical Project

To the left you will find the overhead light plot for The Whipping Man project for Advanced Lighting II. The theater for this project is the Raymond Hodges Theatre at VCU and the lighting inventory was restricted to the VCU inventory.

Below are other elements that were required for this assignment, such as a Color Angle Concept, Magic Sheet, paperwork package including Channel Hookup, Instrument Schedule, Color Count, and Gobo Count, as well as three 3D renderings done in Augment3d depicting various parts of the play.
